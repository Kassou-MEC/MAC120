Question 2: Relative Frequency Tables

Original Problem: Complete a relative frequency table and answer questions about the data.

ScoreRelative Frequency

Find the missing relative frequency.

Solution:

Sum of known relative frequencies = 0.12 + 0.18 + 0.24 + 0.16 = 0.70

Missing relative frequency = 1 - 0.70 = 0.30

Exercise 2.2

Using the completed table from Exercise 2.1:

a) How many students in the sample had a score of at least 70?

b) If there are 600 students total, estimate how many scored between 60 and 89.

Solution:

a) Relative frequency for scores ≥ 70 = 0.30 + 0.24 + 0.16 = 0.70

Number of students = 0.70 × 50 = 35 students

b) Relative frequency for scores 60-89 = 0.18 + 0.30 + 0.24 = 0.72

Estimated number = 0.72 × 600 = 432 students

Question 3: Stem-and-Leaf Displays and Five-Number Summary

Original Problem: Interpret a stem-and-leaf display and calculate the five-number summary.

Exercise 3.1

Consider the following stem-and-leaf display:

StemLeaf
23 5 7
31 2 4 6 8
40 2 5 7 9
51 3 6

Write down the raw data represented by this display.

Solution:

23, 25, 27, 31, 32, 34, 36, 38, 40, 42, 45, 47, 49, 51, 53, 56

Exercise 3.2

Using the data from Exercise 3.1, obtain the five-number summary (Min, Q1, Q2, Q3, Max).

Solution:

Data in order: 23, 25, 27, 31, 32, 34, 36, 38, 40, 42, 45, 47, 49, 51, 53, 56

n = 16

Min = 23

Q1 position = (16+1)/4 = 4.25 → value between 4th and 5th: (31+32)/2 = 31.5

Median (Q2) position = (16+1)/2 = 8.5 → value between 8th and 9th: (38+40)/2 = 39

Q3 position = 3×(16+1)/4 = 12.75 → value between 12th and 13th: (47+49)/2 = 48

Max = 56

Question 4: Five-Number Summary and Box Plots

Original Problem: Calculate the five-number summary for a dataset.

Exercise 4.1

Consider the following data set representing the weights (in pounds) of 15 packages:

12, 15, 18, 20, 22, 24, 25, 26, 28, 30, 32, 35, 38, 40, 45

Obtain the five-number summary for this data.

Solution:

Data is already ordered: 12, 15, 18, 20, 22, 24, 25, 26, 28, 30, 32, 35, 38, 40, 45

n = 15

Min = 12

Q1 position = (15+1)/4 = 4 → 4th value = 20

Median (Q2) position = (15+1)/2 = 8 → 8th value = 26

Q3 position = 3×(15+1)/4 = 12 → 12th value = 35

Max = 45

Exercise 4.2

The following data represents the number of minutes 20 customers waited in line:

5, 8, 10, 12, 15, 18, 20, 22, 25, 28, 30, 32, 35, 38, 40, 42, 45, 48, 50, 55

Calculate the five-number summary for this data.

Solution:

Data is ordered: 5, 8, 10, 12, 15, 18, 20, 22, 25, 28, 30, 32, 35, 38, 40, 42, 45, 48, 50, 55

n = 20

Min = 5

Q1 position = (20+1)/4 = 5.25 → value between 5th and 6th: (15+18)/2 = 16.5

Median (Q2) position = (20+1)/2 = 10.5 → value between 10th and 11th: (28+30)/2 = 29

Q3 position = 3×(20+1)/4 = 15.75 → value between 15th and 16th: (40+42)/2 = 41

Max = 55

Five-number summary: 5, 16.5, 29, 41, 55

Exercise 4.3

Given the five-number summary for a dataset: Min=15, Q1=22, Median=30, Q3=38, Max=50

a) Calculate the Interquartile Range (IQR)

b) Identify any potential outliers using the 1.5×IQR rule

Solution:

a) IQR = Q3 - Q1 = 38 - 22 = 16

b) Lower fence = Q1 - 1.5×IQR = 22 - 1.5×16 = 22 - 24 = -2

Upper fence = Q3 + 1.5×IQR = 38 + 1.5×16 = 38 + 24 = 62

Since all values (15 to 50) are between -2 and 62, there are no outliers.
